Categories of Cyber security

Application security

Focuses on keeping software devices free of threats, it is the process of developing, adding and testing security features within applications.

Network security

The practice of securing computer networks from intruders

Operational security

Risk management process used to discover potential threats and vulnerabilities in software and hardware.

Informational security

Protects the integrity of data, in both storage and transit.

End-user education

Addressing the human end of software (users), teaching users good security practice.
