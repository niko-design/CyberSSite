Types Of Cyber Threats

Backdoors

Hackers use or create backdoors to gain remote access to systems.

Malware

Software that is designed specifically to disrupt, damage, or gain unauthorised access to a computer system examples of this are ransomware, Trojans, spyware, and worms.

Formjacking

Formjacking is where hackers insert malicious code into online forms.

DDOS attacks

Distributed denial of service commonly know as DDOS attack is when servers, service, or network servers are flooded with an overwhelming amount of internet traffic knocking them offline.

Cryptojacking

cryptojacking is where hackers install illicit crypto mining software onto another party's computing resources allowing hackers to mine Crypto without paying using other people resources.
